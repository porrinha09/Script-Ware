# Instalação:

1- Git clone
```sh
git clone https://github.com/porrinha09/Script-Ware.git
```

2- cd
```sh
cd Script-Ware
```

3- Permissão
```sh
chmod +x Script-Ware
```

4- Inciar
```sh
./Script-Ware
```
